# Tennis-2024
Inferential Modeling of Men’s Tennis Performance in 2024

Inferential analysis of the dataset containing all men’s tennis matches played in 2024. The project includes analysis of the variables (considered individually, in pairs, and in groups), time series, spatial series (with accompanying choropleth maps), spatio-temporal series, empirical distribution functions, linear regression models, and local linear regression models.
All statistical analyses were performed in R, and the final document was produced in LaTeX.


## Repository Structure

- `Code/` — main analysis code
- `Data/` — data folder
   - `atp_matches_2024.csv` — raw data (source: link)
   - `Metadata.txt` — variable descriptions
- `graphics/` — output plots
- `Report tennis 2024` — final report


### Dataset
The dataset used in this project was obtained from
